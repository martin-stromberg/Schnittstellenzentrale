# Impressum

Die Impressum-Seite stellt den Inhalt einer Markdown-Datei (`impressum.md`) als formatierte HTML-Seite unter `/impressum` bereit. Das Feature ist dateigesteuert optional: Existiert die Datei nicht, ist die Seite unsichtbar — weder der Footer-Link noch der Seiteninhalt werden angezeigt. Sobald die Datei abgelegt wird, ist das Impressum ohne Neustart oder Konfigurationsänderung aktiv.
