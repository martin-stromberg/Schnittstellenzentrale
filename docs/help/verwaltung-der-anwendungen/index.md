# Verwaltung der Anwendungen

Die Funktion ermöglicht das Anlegen von Anwendungsgruppen (`ApplicationGroup`) und Anwendungen (`Application`) direkt in der Benutzeroberfläche. Damit werden die in der linken Seitenleiste angezeigten Datensätze befüllbar, ohne dass ein direkter Datenbankzugriff notwendig ist.
