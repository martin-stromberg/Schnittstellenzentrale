# Verwaltung der Anwendungen — Ablauf für Anwender

## Voraussetzungen

Die Anwendung ist geöffnet und die Startseite ist sichtbar. Die linke Seitenleiste zeigt die Schaltflächen **Neue Gruppe** und **Neue Anwendung**.

---

## Neue Gruppe anlegen

### 1. Formular öffnen

Klicken Sie in der linken Seitenleiste auf **Neue Gruppe**. Das Formular „Neue Gruppe" wird direkt unterhalb der Schaltflächen eingeblendet.

### 2. Name eingeben

Geben Sie im Feld **Name** den gewünschten Namen der Gruppe ein. Das Feld ist ein Pflichtfeld.

### 3. Speichern

Klicken Sie auf **Speichern**. Die Gruppe wird angelegt, das Formular schließt sich, und die Gruppenliste in der Seitenleiste wird aktualisiert.

> **Hinweis:** Ist das Feld „Name" leer, erscheint eine Fehlermeldung direkt neben dem Feld. Die Gruppe wird erst gespeichert, wenn alle Pflichtfelder ausgefüllt sind.

### Abbrechen

Klicken Sie auf **Abbrechen**, um das Formular ohne Speichern zu schließen.

---

## Neue Anwendung anlegen

### 1. Formular öffnen

Klicken Sie in der linken Seitenleiste auf **Neue Anwendung**. Das Formular „Neue Anwendung" wird direkt unterhalb der Schaltflächen eingeblendet.

### 2. Pflichtfelder ausfüllen

Füllen Sie die folgenden Pflichtfelder aus:

- **Name** — Anzeigename der Anwendung in der Seitenleiste.
- **Basis-URL** — Basisadresse des Dienstes (z. B. `https://intern/meinservice`).

### 3. Optionale Felder ausfüllen (nach Bedarf)

- **Beschreibung** — Kurze Erläuterung zum Zweck der Anwendung.
- **Swagger-URL** — URL zur Swagger/OpenAPI-Beschreibung der Anwendung.
- **Metadaten-URL** — URL zu weiteren Metadaten (z. B. OData-Metadatenendpunkt).
- **Gruppe** — Wählen Sie eine vorhandene Gruppe aus der Liste. Die Option **Ohne Gruppe** belässt die Anwendung außerhalb jeder Gruppe.

### 4. Speichern

Klicken Sie auf **Speichern**. Die Anwendung wird angelegt, das Formular schließt sich, und die Seitenleiste wird aktualisiert. Im Benutzermodus wird die Anwendung automatisch Ihrem Konto zugeordnet.

> **Hinweis:** Sind „Name" oder „Basis-URL" leer, erscheint eine Fehlermeldung direkt neben dem jeweiligen Feld.

### Abbrechen

Klicken Sie auf **Abbrechen**, um das Formular ohne Speichern zu schließen.

---

## Ergebnis

Die neu angelegte Gruppe erscheint als aufklappbarer Bereich in der Seitenleiste. Die neu angelegte Anwendung erscheint als Schaltfläche innerhalb der zugeordneten Gruppe oder im Bereich **Ohne Gruppe**. Per Klick auf eine Anwendung öffnet sich deren Detailansicht im Hauptbereich.
