# Schnittstellenzentrale

Die Schnittstellenzentrale ist eine Blazor Server-Anwendung zur zentralen Verwaltung lokaler Webservice-Endpunkte. Sie ermöglicht das Anlegen, Gruppieren und direkte Ausführen von HTTP-Endpunkten mit verschiedenen Authentifizierungstypen, den Import von Endpunkten aus Swagger/OpenAPI- und OData-Definitionen sowie Echtzeit-Synchronisation im Teammodus über SignalR. Die Oberfläche ist in drei Bereiche gegliedert (Workspaces, Environments, History) und basiert auf der Komponentenbibliothek ShadcnBlazor.
