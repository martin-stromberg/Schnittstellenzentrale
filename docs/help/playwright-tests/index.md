# Playwright-Tests

Die Playwright-Testinfrastruktur ergänzt das Testprojekt `Schnittstellenzentrale.Tests` um End-to-End-UI-Tests. Die Tests starten einen echten Kestrel-Server, steuern einen Chromium-Browser und prüfen die wichtigsten Kernabläufe der Anwendung vollautomatisch.
