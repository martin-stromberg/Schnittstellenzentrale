# Systemumgebungen

Systemumgebungen erlauben es, benannte Variablensätze zu definieren und einer aktiven Umgebung zuzuordnen. Variablen werden in Endpunkten als `{{variablenname}}`-Platzhalter verwendet und beim Absenden einer Anfrage automatisch aufgelöst — ohne die gespeicherten Konfigurationsdaten zu verändern. So können unterschiedliche Zielumgebungen (z. B. Entwicklung, Test, Produktion) ohne Anpassung der Endpunkte genutzt werden.
