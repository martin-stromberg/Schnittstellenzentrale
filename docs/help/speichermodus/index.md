# Speichermodus

Die Anwendung unterscheidet zwischen zwei Speichermodi: **Team** für geteilte Teamdaten und **Benutzer** für benutzerspezifische Daten. Der Benutzer wählt den gewünschten Modus in der Titelleiste; die Auswahl wird im Browser gespeichert und beim nächsten Anwendungsstart automatisch wiederhergestellt.
