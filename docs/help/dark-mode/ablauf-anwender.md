# Dark Mode — Ablauf für Anwender

## Voraussetzungen

Keine besonderen Voraussetzungen. Der Theme-Umschalter ist jederzeit in der Titelleiste der Anwendung verfügbar.

## Schritt-für-Schritt-Anleitung

### 1. Anwendung öffnen

Beim Öffnen der Anwendung wird automatisch das zuletzt gewählte Farbschema wiederhergestellt. Wer die Anwendung zum ersten Mal benutzt, sieht das helle Layout.

### 2. Farbschema wechseln

In der Titelleiste rechts befindet sich der Theme-Umschalter:

- **Helles Layout aktiv:** Der Umschalter zeigt ein Mond-Symbol (`☽`). Ein Klick darauf aktiviert den dunklen Modus.
- **Dunkles Layout aktiv:** Der Umschalter zeigt ein Sonnen-Symbol (`☀`). Ein Klick darauf kehrt zum hellen Modus zurück.

Die gesamte Anwendung wechselt sofort das Erscheinungsbild — Navigation, Inhaltsbereich und alle Dialoge.

> **Hinweis:** Der Umschalter befindet sich in derselben Zeile wie das Modus-Auswahlfeld (Team / Benutzer), ganz rechts in der Titelleiste.

### 3. Einstellung wird gespeichert

Die gewählte Einstellung bleibt auch nach dem Schliessen des Browsers erhalten. Beim nächsten Aufruf der Anwendung wird dasselbe Farbschema automatisch wiederhergestellt.

> **Hinweis:** Die Einstellung ist browserspezifisch. Wer die Anwendung in einem anderen Browser oder auf einem anderen Gerät öffnet, sieht zunächst wieder das helle Layout.

## Ergebnis

Die gesamte Anwendung wird im gewählten Farbschema dargestellt. Die Einstellung bleibt dauerhaft gespeichert, bis der Benutzer sie erneut ändert oder den Browser-Cache leert.
