# Dark Mode

Die Anwendung unterstützt ein schaltbares Farbschema (Hell/Dunkel). Der Benutzer wählt sein bevorzugtes Theme per Klick auf einen Umschalter in der Titelleiste; die Auswahl wird im Browser gespeichert und bei jedem Aufruf automatisch wiederhergestellt.
