## Instructions

An Imprintum is a legal requirement in many countries for websites and online services. It typically includes information about the owner of the website, contact details, and other relevant legal information.
It's only necessary to have an Imprintum if the website or online service is considered a commercial activity or if it collects personal data from users. If the website is purely informational and does not engage in any commercial activities, an Imprintum may not be required.


## To Do

If the imprintum is necessary, the content must be stored in the file impressum.md. Translations can be stored in alternative files with the extension .de.md or .en.md. The Schnittstellenzentrale will display the appropriate translation based on the user's language settings. If no translation is available, the default file impressum.md will be displayed.

If no imprintum is necessary, the file can be deleted.
